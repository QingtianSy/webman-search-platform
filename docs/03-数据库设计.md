# 数据库设计

## 1. 存储职责划分

| 存储引擎 | 存储内容 | 选型理由 |
|---------|---------|---------|
| MySQL 8.0 | 用户、权限、订单、配置等结构化数据 | ACID 事务、关系查询、成熟稳定 |
| MongoDB 7 | 题目、采集原始数据、日志详情 | 灵活 Schema、大字段、高写入 |
| Elasticsearch 8 | 题目搜索索引、日志搜索索引 | 倒排索引、全文检索、毫秒响应 |
| Redis 7.2 | Token、权限缓存、额度、限流 | 内存操作、原子性、高并发 |

## 2. MySQL 表设计

### 2.1 认证与权限（RBAC）

#### users - 用户表
```sql
CREATE TABLE users (
    id              BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    username        VARCHAR(64)  NOT NULL UNIQUE,
    password_hash   VARCHAR(255) NOT NULL,
    nickname        VARCHAR(64)  NOT NULL DEFAULT '',
    email           VARCHAR(128) NOT NULL DEFAULT '',
    phone           VARCHAR(20)  NOT NULL DEFAULT '',
    avatar          VARCHAR(255) NOT NULL DEFAULT '',
    default_portal  VARCHAR(20)  NOT NULL DEFAULT 'user' COMMENT 'user|admin',
    status          TINYINT      NOT NULL DEFAULT 1 COMMENT '1=正常 0=禁用',
    last_login_at   DATETIME     NULL,
    last_login_ip   VARCHAR(45)  NOT NULL DEFAULT '',
    created_at      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_status (status),
    INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### roles - 角色表
```sql
CREATE TABLE roles (
    id          INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    code        VARCHAR(32)  NOT NULL UNIQUE COMMENT 'user|vip_user|operator|admin|super_admin',
    name        VARCHAR(64)  NOT NULL,
    description VARCHAR(255) NOT NULL DEFAULT '',
    sort        INT          NOT NULL DEFAULT 0,
    status      TINYINT      NOT NULL DEFAULT 1,
    created_at  DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at  DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### permissions - 权限表
```sql
CREATE TABLE permissions (
    id          INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    code        VARCHAR(64)  NOT NULL UNIQUE COMMENT 'portal.access|search.query|admin.access',
```

#### menus - 菜单表
```sql
CREATE TABLE menus (
    id              INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
```

## 5. Redis Key 设计

| Key 模式 | 类型 | TTL | 用途 |
|---------|------|-----|------|
| `token:user:{user_id}` | string | 7d | JWT Token |
| `perm:user:{user_id}` | set | 1h | 用户权限码集合 |
| `menu:user:{user_id}` | string(json) | 1h | 用户菜单树 |
| `api_key:{api_key}` | hash | 1h | API Key 信息 |
| `quota:user:{user_id}` | string(int) | - | 用户剩余额度 |
| `quota:key:{api_key_id}` | string(int) | - | API Key 剩余额度 |
| `rate:user:{user_id}:{api}` | string(int) | 60s | 用户接口限流计数 |
| `rate:ip:{ip}:{api}` | string(int) | 60s | IP 接口限流计数 |
| `search:result:{md5}` | string(json) | 10m | 搜索结果缓存 |
| `hot:search` | zset | - | 热搜排行榜 |
| `collect:task:{task_id}` | hash | - | 采集任务实时状态 |
| `lock:quota:{user_id}` | string | 5s | 额度扣减分布式锁 |
