## 4. 宝塔面板部署步骤

### 4.1 安装服务
```bash
# 通过宝塔面板安装
- Nginx
- PHP 8.2（安装扩展：swoole, redis, mongodb, pdo_mysql, curl, mbstring）
- MySQL 8.0
- Redis 7.2
- MongoDB 7（手动安装或通过宝塔软件商店）
```

### 4.2 部署后端
```bash
cd /var/www/search-platform
git clone https://github.com/QingtianSy/webman-search-platform.git .

cd backend
composer install --no-dev --optimize-autoloader

cp .env.production.example .env
# 编辑 .env 填写数据库连接信息
```

### 4.3 环境变量配置
```bash
# .env
APP_NAME=webman-search-platform
APP_ENV=prod
APP_DEBUG=false

MYSQL_HOST=127.0.0.1
MYSQL_PORT=3306
MYSQL_DATABASE=search_platform
MYSQL_USERNAME=search_platform
MYSQL_PASSWORD=your_password

MONGO_URI=mongodb://127.0.0.1:27017
MONGO_DATABASE=search_platform

ES_HOST=http://127.0.0.1:9200
ES_USERNAME=
ES_PASSWORD=

REDIS_HOST=127.0.0.1
REDIS_PORT=6379
REDIS_PASSWORD=your_redis_password

JWT_SECRET=your_random_secret_key_at_least_32_chars
JWT_EXPIRE=604800
```

### 4.4 Systemd 服务
```ini
# /etc/systemd/system/webman-search-platform.service
[Unit]
Description=Webman Search Platform
After=network.target mysql.service redis.service mongod.service

[Service]
Type=simple
User=www-data
Group=www-data
WorkingDirectory=/var/www/search-platform/backend
ExecStart=/usr/bin/php start.php start
ExecReload=/usr/bin/php start.php reload
ExecStop=/usr/bin/php start.php stop
Restart=always
RestartSec=5
LimitNOFILE=65536

[Install]
WantedBy=multi-user.target
```

```bash
systemctl daemon-reload
systemctl enable webman-search-platform
systemctl start webman-search-platform
systemctl status webman-search-platform
```

### 4.5 Nginx 配置
```nginx
# /etc/nginx/sites-available/search-platform.conf
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态资源
    location / {
        root /var/www/search-platform/frontend/dist;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /api/ {
        proxy_pass http://127.0.0.1:8787;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location /open/ {
        proxy_pass http://127.0.0.1:8787;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location /health {
        proxy_pass http://127.0.0.1:8787;
    }

    location /ready {
        proxy_pass http://127.0.0.1:8787;
    }
}
```

### 4.6 初始化数据库
```bash
cd /var/www/search-platform/backend

# 执行 MySQL 迁移
php database/migrations/run.php

# 导入初始数据
php database/seeds/run.php
```

### 4.7 构建前端
```bash
cd /var/www/search-platform/frontend
pnpm install
pnpm build:naive
# 构建产物在 apps/web-naive/dist/
cp -r apps/web-naive/dist/* /var/www/search-platform/frontend/dist/
```

## 6. 日志管理

| 日志文件 | 路径 | 说明 |
|---------|------|------|
| Webman 应用日志 | `runtime/logs/webman.log` | 应用级日志（Monolog） |
| Workerman 进程日志 | `runtime/logs/workerman.log` | Worker 进程日志 |
| stdout 日志 | `runtime/logs/stdout.log` | 标准输出 |
| Nginx access | `/var/log/nginx/access.log` | HTTP 访问日志 |
| Nginx error | `/var/log/nginx/error.log` | Nginx 错误日志 |

日志切割：配置 logrotate 每日切割，保留 7 天。

## 7. 备份策略

| 数据 | 频率 | 保留 | 方式 |
|------|------|------|------|
| MySQL | 每日 | 7 天 | mysqldump + crontab |
| MongoDB | 每日 | 7 天 | mongodump + crontab |
| .env | 手动 | 永久 | 加密存储 |
| 上传文件 | 每周 | 30 天 | rsync |

---

## 8. 安全要求

- ❌ Redis / MongoDB / Elasticsearch 不暴露公网
- ❌ .env 文件不提交到 Git
- ❌ 密码不明文存储
- ✅ 所有数据库仅监听 127.0.0.1
- ✅ Nginx 配置 HTTPS（Let's Encrypt）
- ✅ Webman 仅监听 127.0.0.1:8787
- ✅ 定期更新系统和依赖
