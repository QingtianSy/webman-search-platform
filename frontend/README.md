# frontend

统一前端项目骨架，采用：
- 一个登录页
- 一个前端项目
- 统一用户体系
- 按角色和权限动态决定进入用户端或管理端模块

## 当前阶段
前端已进入 **更完整的管理系统页面细化阶段**：
- 用户侧文档中心支持分类与详情展示
- 用户侧采集任务支持详情查看
- 管理侧用户/角色/权限/菜单/套餐页面已具备基础详情交互
- 统一布局、认证状态、动态菜单预留结构已成型

## 当前能力
- 统一登录
- 工作台卡片页
- 题目管理表格/交互页
- 搜题日志表格页
- API Key 表格页
- 钱包套餐卡片页
- 文档中心页
- 采集任务页
- 管理端用户/角色/权限/菜单/套餐/公告/系统配置/文档/采集管理页
